select_client(use_service_role: bool = False) → SyncClient[source]¶

Selects the appropriate Supabase client based on the use_service_role parameter.

Parameters:

use_service_role (bool, optional) – Determines whether to use the service role client or the default client. Defaults to False.

Returns:

The appropriate Supabase client.

Return type:

Client

Notes

Service role client should only be used for operations where a new

user row is being inserted. Otherwise use default client for RLS policy on authenticated user.

If service_client is None – no service role key was provided –

the default_client is returned.
