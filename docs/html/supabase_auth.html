reset_password(*, email: str, domain: str) → None[source]¶

Resets the password for a user with the provided email.

Parameters:

  • email (str) – The email of the user.

  • domain (str) – The domain of the application.

Raises:

AuthInvalidCredentialsError – If an error occurs during the password reset process.

Example
